# Uber-PowerBI-Dashboard-

📊 Project Description
A comprehensive Uber Business Intelligence Dashboard built in Power BI to analyze rider behavior, trip revenue, vehicle performance, and operational metrics across locations and customer ratings. This interactive dashboard provides actionable insights through advanced visualizations and dynamic filters, enabling data-driven decision-making for Uber's operations management.

🎯 Key Features & Capabilities
Multi-Page Power BI Report: Overview, Vehicle, Rider, Revenue, and Location pages

Interactive Power BI Filters: Slicers for vehicle type, date ranges, payment methods, and locations

KPI Tracking: Completed/lost bookings, revenue, distance traveled, average distance

DAX Calculations: Custom measures for YoY growth, contribution percentages, and performance metrics

Performance Analysis: Driver & customer ratings, pickup/drop hotspots, cancellation reasons

Revenue Breakdown: By vehicle type, payment method, customer segment, and time period
